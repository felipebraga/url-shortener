## Getting Started

### Prerequisites

- [JDK 21](https://openjdk.org/projects/jdk/21/)
- [SDKMAN!](https://sdkman.io/)
- [Maven](https://maven.apache.org/)
- [Docker](https://www.docker.com/products/docker-desktop/)

### Usage

At the root directory run:

```
docker compose up
```

The database is running.
